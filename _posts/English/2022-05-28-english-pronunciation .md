# 自然拼音

## 26个字母

### 元音和辅音
元音：A E I O U，半元音：Y

元音会有多种发音，辅音只有1种发音

> 为什么会有元音和辅音之分？
```
中文：一声一韵，前声后韵，如：李li
英文：一辅一元，前辅后元，如：ba \bæ\ bag, bat
```

> 两个相同的辅音，只读一个音
```
rabbit \'ræbit\
```
